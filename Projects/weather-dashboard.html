A user friendly, quick app to check for current weather conditions as well as 5-day forecasts. Large, weather icons indicate the overall weather conditions. All basic stats are listed neatly for each date. The current UV index is listed with a color guide to indicate where it falls on the UV scale. Easily search the weather by city. Your past search history is stored in a clickable list to easily access for future reference.

This website utilizes the Open Weather API. HTML, CSS, and Javascript were used to execute the site in a simple and easy to navigate interface. A blue and gray color template and sans serif font was chosen to keep with the themes generally used in weather apps.
